# Frontend Assignment Checkout Screen

React JS implementation of the checkout/billing screen from the provided Figma screenshot.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Built with React components in `src/App.jsx`.
- Styling is contained in `src/styles.css`.
- Desktop measurements follow the provided card, header, and column dimensions.
- The layout is responsive and stacks cleanly on tablet/mobile widths.
